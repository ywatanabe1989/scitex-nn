scitex-nn
=============

PyTorch neural-network building blocks for scientific signal processing —
filters, spectral transforms (Wavelet, Hilbert, PSD, Spectrogram),
phase-amplitude coupling (PAC, ModulationIndex), attention, and ready-made
architectures (BNet, MNet, ResNet1D) — extracted from the SciTeX ecosystem as
a standalone package.
